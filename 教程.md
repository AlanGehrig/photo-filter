# Photo Filter Pro 使用教程

**摄影师级智能照片筛选工具**

---

## 快速开始

### 1. 运行程序

双击 `run_gui.py` 或在终端运行：
```bash
python run_gui.py
```

### 2. 选择照片文件夹

点击「📂 浏览」按钮，选择你要筛选的照片所在文件夹。

### 3. 选择输出目录

点击「📤 选择」，选择处理后照片的保存位置。

---

## 筛选功能

### 格式筛选
- 支持：JPG, PNG, WebP, RAW, HEIC
- 默认：全部格式

### 尺寸筛选
- 拖动滑块设置最小文件尺寸（0-50 MB）
- 用于过滤截图或小图

### 质量筛选（AI功能）

| 功能 | 说明 |
|------|------|
| 🔵 模糊检测 | 自动识别并标记模糊照片 |
| ☀️ 曝光检测 | 识别过曝/欠曝照片 |
| 👤 人脸检测 | 筛选包含人脸的照片 |
| ⚡ 连拍检测 | 识别同一场景的连拍照片 |

---

## 使用流程

```
1. 选择文件夹 → 2. 设置筛选条件 → 3. 点击"应用筛选" → 4. 查看结果 → 5. 开始处理
```

### 筛选照片
1. 设置筛选条件
2. 点击「🔍 应用筛选」
3. 在右侧查看筛选结果

### 处理照片
1. 在列表中选择要处理的照片（可多选）
2. 点击「🚀 开始处理」
3. 等待处理完成

---

## 快捷操作

- **全选**: 点击「全选」按钮
- **取消选择**: 点击「取消」按钮
- **预览**: 双击照片预览
- **拖拽**: 直接拖拽文件夹到窗口

---

## 常见问题

### Q: 程序打不开？
A: 确保已安装依赖：
```bash
pip install -r requirements.txt
```

### Q: 没有看到照片？
A: 检查文件夹路径是否正确，确保文件夹中有支持格式的照片。

### Q: 筛选结果为空？
A: 尝试放宽筛选条件，或检查输出目录是否有写入权限。

- **AI模型**: PyTorch (可选)
